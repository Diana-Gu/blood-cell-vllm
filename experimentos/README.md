# Auditoría cruzada Gemini vs Qwen2.5-VL

Se realizó una evaluación independiente sobre una muestra
estratificada de 200 imágenes con el objetivo de estimar
la concordancia entre dos evaluadores multimodales.

Modelos utilizados:

- Gemini-3-flash-preview
- Qwen2.5-VL

Objetivo:

- Evaluar la robustez del proceso de auditoría automática.
- Analizar coincidencias y discrepancias entre evaluadores.

Importante:
Este experimento no forma parte de la comparativa principal
de modelos generadores del TFM y se utiliza únicamente como
mecanismo adicional de validación.
